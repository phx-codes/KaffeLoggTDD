# Blockchain Kaffelogg - TDD Projekt

Detta är ett Node.js-API byggt med Express och testat med Vitest och Supertest. Projektet använder Test-Driven Development (TDD).

Här är länkarna till tre commits i historiken som visar att testerna skrevs innan produktionskoden:

3. **Test för Blockchain-klassen (Röd):** [länk](https://github.com/phx-codes/KaffeLoggTDD/commit/de3ab72bfd5ead4d2319b6a8dfccf5aba7564a18)

## Hur man kör projektet
1. Installera dependencies: `npm install`
2. Kör testerna: `npm run test`
3. Starta servern: `node server.js`
